# Wheelie Golf Prototype (Phaser 3 + Vite)

Single-input wheelie golf cart prototype targeting web (desktop + mobile browsers).

## Run

```bash
npm install
npm run dev
```
Then open the printed local URL (Vite dev server).

## Build

```bash
npm run build
npm run preview
```

## Controls
- Desktop: Hold spacebar or hold mouse click anywhere to throttle.
- Mobile: Touch and hold anywhere.

Fail states: front wheel touches ground/hazard, stall meter fills when you stop throttling, or over-rotation flip.
